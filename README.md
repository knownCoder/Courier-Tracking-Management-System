# 📦 Courier Tracking System

A modern Courier Tracking System built with Flask, SQLAlchemy, SQLite, and Bootstrap.

This application allows administrators to manage shipments and customers to track their packages in real time.

---

## 🚀 Features

### Customer Features
- Track shipment using Tracking ID
- View shipment details
- Shipment status updates
- Tracking timeline
- Responsive Bootstrap UI

### Admin Features
- Admin Login
- Admin Dashboard
- Add New Shipment
- Update Shipment Status
- Update Shipment Location
- View All Shipments
- Dashboard Analytics Cards

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/courier-tracking-system.git
```

### Move into Project

```bash
cd courier-tracking-system
```

### Create Virtual Environment

```bash
python3 -m venv venv
```

### Activate Virtual Environment

Linux/Mac:

```bash
source venv/bin/activate
```

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python3 app.py
```
